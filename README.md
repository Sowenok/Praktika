## Вредные советы для непослушных детей 
## Григорий О.



Начиная драку с папой,\
Затевая с мамой бой,\
Постарайся сдаться маме,\
Папа пленных не берет.\
Кстати, выясни у мамы:\
Не забыла ли она -\
Пленных бить ремнем по попе\
Запрещает Красный Крест\.
